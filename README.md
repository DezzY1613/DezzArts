# Что это?
## Полностью рабочая программа на C++, которая рисует в консоли 6 анимированных шедевров:

- Волна-змея (чередует направление)
- Смайлик
- Сердечко
- Шахматная доска
- Логотип Windows
- Клетка

# Как запустить

Скачай все файлы в одну папку
Открой в Visual Studio 2022
Запусти от имени администратора (не обязательно)
Введи номер анимации (0–5)
Наслаждайся

# Требования

Windows (любой)
Visual Studio (или любой C++ компилятор)
5 минут свободного времени и желание удивиться
